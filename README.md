## Deskripsi Aplikasi
Aplikasi Manajemen Pemain Sepak Bola merupakan program berbasis Command Line
Interface (CLI) yang digunakan untuk mengelola data pemain dalam sebuah tim sepak
bola. Aplikasi ini dibuat untuk membantu proses pencatatan data pemain secara
sederhana, terstruktur, dan mudah digunakan.

## Fitur Aplikasi
- Menu utama berbasis perulangan (loop)
- Menambah data pemain (Create)
- Menampilkan seluruh data pemain (Read)
- Memperbarui data pemain berdasarkan ID (Update)
- Menghapus data pemain berdasarkan ID (Delete)
- Validasi input untuk mencegah kesalahan program

## User Guide
1. Jalankan program menggunakan Python.
2. Pilih menu 1 untuk menambahkan data pemain sepak bola.
3. Pilih menu 2 untuk melihat seluruh data pemain.
4. Pilih menu 3 untuk memperbarui data pemain berdasarkan ID.
5. Pilih menu 4 untuk menghapus data pemain berdasarkan ID.
6. Pilih menu 5 untuk keluar dari aplikasi.
